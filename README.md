# TV2 Region Designsystem Starter

Simple static setup with:

- `index.html` (frontpage)
- `article.html` (article page)
- `design-manual.html` (design manual page)
- Shared theme switcher on all pages
- `HOW-TO.md` (task execution workflow and skill usage guide)

## Themes included

- `tv2Oj`
- `tv2Nord`
- `tv2Syd`
- `tv2Fyn`
- `tv2East`
- `kosmopol`

## Run

You can open the HTML files directly in a browser, or run a tiny local server:

```bash
cd /Users/frsc/mcp-projekter/tv2reg-designsystem
python3 -m http.server 8080
```

Then open `http://localhost:8080`.
